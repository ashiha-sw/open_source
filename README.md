# cprog
## 사용할 언어는 c언어
***
_곱셉을 해주는 알고리즘_
***
```c
#include <stdio.h>
 
int main(void)
{
  int i,j;
  for(i = 0; i<=9; i++)
    for(j = 0; j<=9; j++)
      printf("%d 곱하기 %d은 %d\n",i,j,i*j);

  return 0;
}
```
 ***
 **for문을 반복적으로 사용할때 어떻게 되는지 확실히 알수있음**
